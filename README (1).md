### Launch Friendliness Score Methodology
The score is a composite metric weighted across four dimensions:

```
Survival Rate 30D    → 35% weight
Low Sniper Exposure  → 25% weight
Low Launch Cost      → 20% weight
Trader Quality       → 20% weight
```

### Key Insight
> Base wins the overall score at 71 despite having fewer launches than Solana. Solana's near-zero cost advantage is offset by a 78.1% sniper rate — the highest of all three chains. Ethereum scores lowest at 44, combining the highest cost with the worst survival rate.

### Solana Bonding Curve Journey
```
Alive 30D:    13,800 tokens  (graduated + still trading)
Ghost:         4,800 tokens  (zero activity)
```

When a Base token survives, it generates 5x the volume of a Solana survivor and 20x that of an Ethereum survivor. Survival quality matters as much as survival rate.

**Solana Bonding Curve Filter:**
4,500 tokens (24.48% of Solana launches) failed before graduating from Pump.fun's bonding curve to a real DEX. The bonding curve acts as a natural but brutal filter.

## 🔢 Key DAX Measures

```dax
Total Launches =
SUM(Total_Token_Launch[new_tokens_launched])

Ghost Rate % =
DIVIDE(
    CALCULATE(SUM(Survival_Rate[token_count]),
    Survival_Rate[Status] = "Ghost"),
    SUM(Survival_Rate[token_count])
)

Survival Rate 30D % =
DIVIDE(
    CALCULATE(SUM(Survival_Rate[token_count]),
    Survival_Rate[Status] IN {"Active","Alive 7D","Alive 30D"}),
    SUM(Survival_Rate[token_count])
)

Avg Launch Cost USD =
ROUND(AVERAGE(Average_Token_Launch[avg_launch_cost_usd]), 6)

Sniper Rate % =
ROUND(AVERAGE(Sniped_Bot[pct_tokens_sniped]), 2)

Launch Friendliness Score =
    (Survival Rate 30D % * 0.35)
    + ((100 - Sniper Rate %) * 0.25)
    + CostScore
    + (MIN(Avg Unique Traders, 100) * 0.20)
```

## ⚠️ Data Notes & Limitations

| Item | Note |
|---|---|
| Date Range | Feb 17 – Mar 18, 2026 (30 days) |
| EVM Launch Detection | Uniswap V3 only — other Base/ETH DEXes not included |
| Solana Launch Detection | Pump.fun + Raydium only — excludes newer launchpads |
| Pump.fun Graduation | `pump_call_withdraw` stopped updating after March 20, 2025. Graduation now detected via DEX trading activity |
| Bot Volume | $1.89B reflects Solana only — EVM bot volume data incomplete |
| Survival Classification | Ghost = zero trades. Dead = had trades but inactive. Alive = traded within window |
| Cost Data | Ethereum costs declining throughout window — averages reflect full 30-day period |
